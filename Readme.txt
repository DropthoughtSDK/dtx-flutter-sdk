Installing Flutter:
	* Follow the instructions in the link to setup flutter development environment in your machine - https://flutter.dev/docs/get-started/install
	* After installing flutter sdk, run flutter doctor to see what dependencies that are required are missing from your machine
	* Install Android studio, android toolchain, vscode, Chrome ( for web app ), XCode for iOS development, etc.
	* Now go to vscode and type 'Flutter' - to create a new project and give it a name
	* Download, clone the files in the \lib folder in bct_dashboard, copy the files to the lib folder in your project and install dependencies in the pubspec.yaml file by running 'flutter pub get'
	* Now the front-end environment is setup
	
Installing NodeJS:
	* Install NodeJS version from https://nodejs.org/en/ globally in the machine
	* Download the files from NodeJS Backend and copy it to your folder
	* Run npm install to install all the required dependencies
	* Use node index.js or nodemon index.js to run the back-end inside localhost:3000
